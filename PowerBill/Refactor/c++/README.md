# Power Billing System (C++ Refactor)

This section is reserved for the **C++ version** of the Power Billing System.  
The goal is to replicate the Java refactor in C++, applying the same object-oriented design principles.

## 📂 Planned Structure
- `Client` → validates and stores client information.
- `Calculator` → applies business rules (tariffs, IVA, Red Cross, fixed fee).
- `Invoice` → generates and formats the billing invoice.
- `Menu` → console interface to process clients and show global report.
- `Main` → entry point of the program.

## 🔮 Current Status
- Work in progress.  
- Classes will mirror the Java implementation, adapted to C++ syntax and best practices.  
- Estimated completion: *soon… or maybe next millennium* ⏱️
