# Power Billing System (Java Refactor)

This project is the **Java refactor** of the original Power Bill program made in PSeInt.  
The goal was to migrate the logic into an **object-oriented design**, separating responsibilities into clear classes.

## 📂 Structure
- `Client` → validates and stores client information.
- `Calculator` → applies business rules (tariffs, IVA, Red Cross, fixed fee).
- `Invoice` → generates and formats the billing invoice.
- `Menu` → console interface to process clients and show global report.
- `Main` → entry point of the program.

## ✅ Current Status
- Fully functional billing system in Java.
- Generates invoices with automatic number and date.
- Produces a **final report** with totals (clients, KWVs, IVA, Red Cross, billed amount).
- Refactored from a single block of code into modular classes.
